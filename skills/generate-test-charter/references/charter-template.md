By the end of this charter session, the tester should:
- Have explored {specific behaviors} under {conditions}

## Test Ideas (Starting Points)

- Try {action} with {condition} and observe {what to watch for}
- Submit {form/action} while {state} is active
- Perform {operation} as {role A}, then verify {effect} as {role B}
- Rapidly {repeat action} and check for {race condition / state corruption}
- Navigate away mid-flow and return — check {session/state persistence}
