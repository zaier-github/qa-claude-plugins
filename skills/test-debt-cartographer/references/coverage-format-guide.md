# Coverage Format Guide

How to extract actionable gap data from common coverage report formats.

---

## lcov (`lcov.info` / `coverage.lcov`)

Text format used by C/C++, Rust, and many JS/TS tools (Istanbul can export it).

**Key fields:**
```
SF:<source file path>          # file being reported
FN:<line>,<function name>      # function definition
FNDA:<hit count>,<function>    # function execution count (0 = never called)
DA:<line>,<hit count>          # line execution count (0 = never executed)
BRDA:<line>,<block>,<branch>,<taken>  # branch coverage
end_of_record
```

**Extract uncovered functions:**
```bash
# Functions with 0 hits
grep "^FNDA:0," lcov.info | sed 's/FNDA:0,//' | sort -u

# Files with any uncovered lines
grep -B50 "DA:[0-9]*,0" lcov.info | grep "^SF:" | sort -u

# Overall summary
grep "^[LBF]H\|^[LBF]F" lcov.info  # LH=lines hit, LF=lines found, BH/BF=branches, FH/FF=functions
```

## Istanbul / NYC (JSON — `coverage-summary.json`)

Standard format for JavaScript/TypeScript projects using Jest, Mocha, NYC, etc.

**Structure:**
```json
{
  "total": { "lines": {"total":100,"covered":67,"pct":67}, "branches": {...}, "functions": {...} },
  "/path/to/file.js": {
    "lines":     { "total": 50, "covered": 32, "pct": 64 },
    "branches":  { "total": 20, "covered":  6, "pct": 30 },
    "functions": { "total": 10, "covered":  7, "pct": 70 },
    "statements":{ "total": 55, "covered": 35, "pct": 63.6 }
  }
}
```

**Extract high-risk gaps:**
```bash
# Files with 0% coverage
python3 -c "
import json
with open('coverage-summary.json') as f:
    data = json.load(f)
for path, stats in data.items():
    if path == 'total': continue
    if stats['lines']['pct'] == 0:
        print(f'0% lines: {path}')
"

# Files with low branch coverage despite reasonable line coverage (Coverage Theater signal)
python3 -c "
import json
with open('coverage-summary.json') as f:
    data = json.load(f)
for path, stats in data.items():
    if path == 'total': continue
    line_pct = stats['lines']['pct']
    branch_pct = stats['branches']['pct']
    if line_pct > 60 and branch_pct < 40:
        print(f'Theater candidate — lines:{line_pct:.0f}% branches:{branch_pct:.0f}%  {path}')
"
```

## pytest-cov (XML — `coverage.xml`)

Generated by `pytest --cov --cov-report=xml`. Based on the Cobertura XML format.

**Structure:**
```xml
<coverage line-rate="0.67" branch-rate="0.45" ...>
  <packages>
    <package name="src.billing">
      <classes>
        <class filename="src/billing/processor.py" line-rate="0.12" branch-rate="0.05">
          <methods>
            <method name="charge_card" hits="0" .../>
          </methods>
          <lines>
            <line number="45" hits="0" branch="true" condition-coverage="0% (0/2)"/>
          </lines>
        </class>
```

**Extract uncovered methods:**
```bash
# Methods with 0 hits
python3 -c "
import xml.etree.ElementTree as ET
tree = ET.parse('coverage.xml')
for cls in tree.iter('class'):
    fname = cls.get('filename','')
    for method in cls.iter('method'):
        if method.get('hits','1') == '0':
            print(f'{fname}::{method.get(\"name\")}')
"

# Files below threshold
python3 -c "
import xml.etree.ElementTree as ET
tree = ET.parse('coverage.xml')
for cls in tree.iter('class'):
    rate = float(cls.get('line-rate', 1))
    branch = float(cls.get('branch-rate', 1))
    if rate < 0.4:
        print(f'Low coverage ({rate:.0%} line, {branch:.0%} branch): {cls.get(\"filename\")}')
"
```

## SimpleCov (Ruby — `coverage/.resultset.json`)

Used by RSpec and Minitest projects via the `simplecov` gem.

**Structure:**
```json
{
  "RSpec": {
    "coverage": {
      "/path/to/app/models/order.rb": {
        "lines": [1, null, 3, 0, 0, null, 1, ...]
      }
    }
  }
}
```

Lines are: integer (hit count), `0` (not covered), or `null` (not executable).

**Extract uncovered files:**
```bash
ruby -e "
require 'json'
data = JSON.parse(File.read('coverage/.resultset.json'))
data.each do |suite, results|
  results['coverage'].each do |file, stats|
    lines = stats['lines'].compact
    covered = lines.count { |l| l > 0 }
    total = lines.count
    pct = total > 0 ? (covered.to_f / total * 100).round : 0
    puts \"#{pct}%  #{file}\" if pct < 40
  end
end
" | sort -n
```

## Go coverage (`coverage.out`)

Generated by `go test -coverprofile=coverage.out`.

**Format:**
```
mode: atomic
github.com/org/repo/pkg/billing/processor.go:45.12,52.3 3 0
#                                              ^line.col   ^ hit count (0=uncovered)
```

**Extract uncovered blocks:**
```bash
# Lines with 0 hits
grep ",0$" coverage.out | awk -F: '{print $1":"$2}' | sort -u

# Generate HTML report for visual inspection
go tool cover -html=coverage.out -o coverage.html
```
